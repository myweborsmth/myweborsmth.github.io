Tip

Use font-size, spacing, and a clear color scale for better readability.

What you can learn

Typography

Choose legible fonts, use a clear scale for headings and body copy, and prefer relative units (rem).

Layout

Master Flexbox & Grid for responsive layouts — start with one column on small screens, multiple on wide screens.

Colors & Contrast

Pick a small palette (primary, accent, neutral). Ensure contrast for accessibility.

About this starter

This demo separates structure (HTML) from presentation (CSS). Use the styles.css file to experiment with spacing, colors and responsive breakpoints. Small incremental changes teach the most.
